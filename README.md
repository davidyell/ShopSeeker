# Shop Seeker

An example application in Larvel to find shops near to a UK postal code.

-   Created using Laravel Breeze as a base.
-   Created in collaboration with Github Copilot.

## Install

-   Clone the repo
-   `composer install`
-   `npm ci`
-   `php artisan migrate:fresh --seed`
